linux 快捷键
1、ls  列出本地址上文件，
    -a    列出所有（包括隐藏文件）
    -l     按照列表方式显示
    -t     按照时间方式排序

2、touch 创建文件 
3、echo   'abc' >  文件名    写入文件内容（覆盖之前的内容）
        echo   'cdc'   >> 文件名      在文件后面添加信息

4、mkdir    -p     test/test1/test2    创建目录
5、cat    文件名    ： 可以查看文件内容
6、head   - (n)  2    文件名   : 可以查看文件的前两行
        tail    -(n) 3  文件名   ： 可以查看文件后几行   (其中n是可以省略的，负号不可以省略)
7、tree   :  可以查看文件数结构
8、cd     :回根目录
            cd  ~   :   回根目录
            cd  -  ： 返回刚才的目录
            cd  ..   :返回上一级目录
            cd  .  : 当前目录

9、find  .   -name  '*.txt'       : 查看当前目录下的所有以 .txt结尾的文件  （name是固定的不用改变）

10、sudo  find   /     -name  '*.txt'     ：使用高权限在根目录下寻找 ，所有以     .txt 为结尾的文件

11、pwd    ：查看当前所在目录

12、cp  ：复制命令
                cp    test1.txt      new.txt
                        原来文件        新文件
                cp    test1.txt     /home/tarena/1910xly/xinwenjainming.txt
                                                         路径加上                    新文件名
13、mv    原文件     新文件    :    剪切
                mv    new.txt    new2.txt

14、rm -f  :强制删除
                -r   :把子目录删除
                -i：询问是否删除文件

15、tar -zcvf   压缩包名称   test.txt  test2.txt   :压缩
         tar   -zxvf  压缩包                                                :解压到当前目录
         tar   -zxvf   压缩包   -C    路径                          : 解压到指定路径

16、chmod  +x    文件名  ： 添加文件的可执行权限
                x  :executable 可执行的
         chmod   777  文件名  

17、 网络操作
            ifconfig    : 查看本机IP地址

             inet 192.168.237.130  这个是本机的ip地址

             ping  + IP / 域名

18、uname   -a  :查看liunx的版本信息

19、添加用户    
            sudo  adduser  +  用户名    ：可以添加用户
                        在  /etc/passwd    文件中最后一行可以看到添加的用户信息


        更改密码
            sudo   passwd     +  用户名   ：直接修改密码
        删除用户
            sudo    deluser   +  用户名     ：删除用户

        修改密码完成后,可以使用如下命令切换到tom用户进行验证：
                su - tom      ：切换到指定用户
